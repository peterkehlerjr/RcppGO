Installation Instructions
*************************

These are basic installation instructions assuming you have administrator rights.

As of version 0.1 the following OS are supported:
- OS X,
- Linux,
- Windows has not yet been tested.

In order to install the package 'RcppGO', 
copy the "install_RcppGO.sh" file to the parent directory of RcppGO
and run it in your terminal. 

You might have to run
chmod +x install_RcppGO.sh
in your terminal.i
