Installation Instructions
*************************

These are generic installation instructions. 

In order to install the package 'RcppGO' properly, 
it is necessary to expose the path to the 'RcppArmadillo' package
and make R aware of other libs to be used.

Without further comments on e.g. OS X (adjust to your os) you should enter this in the terminal:

#
RCPP_CXXFLAGS=`Rscript -e 'Rcpp:::CxxFlags()'`
RCPP_LIBS=`Rscript -e 'Rcpp:::LdFlags()'`

export PKG_CPPFLAGS="-I/home/pkehler/R/x86_64-pc-linux-gnu-library/2.15/RcppArmadillo/include ${RCPP_CXXFLAGS}"
export PKG_LIBS="-larmadillo -llapack  ${RCPP_LIBS}"


R CMD INSTALL RcppGO*

If no errors occurred you should read a message similiar to this:

** R
** preparing package for lazy loading
** help
*** installing help indices
** building package indices ...
** testing if installed package can be loaded
*** arch - i386
*** arch - x86_64

* DONE (RcppGO)


Now the package is installed and ready for use.
