What is RcppGO?

RcppGO is a optimization package for R written in C++. It can be used to solve unconstrained classic or stochastic optimization problems.

What are the key features?

RcppGO

  • is fast due to its implementation in C++
  • is easy to use
  • requires little knowledge about programming
  • is robust
  • ...

How do I try it out?

Download the files in the repository and compile them. An installation instruction is included. So far only Linux and OS X are supported.

Current status

As of Oct 2012 the project is at version 0.1. Updates will follow irregularly.
