What is RcppGO?

RcppGO is a optimization package for R written in C++. It can be used to solve unconstrained classic or stochastic optimization problems.

What are the key features?

RcppGO

  • is fast due to its implementation in C++
  • is easy to use
  • requires little knowledge about programming
  • is robust

Disadvantages

Due to its concept as an stochastic optimization algorithm, the results are also stochastic. Classic optimization problems will converge faster using classic optimization algorithms.

How do I try it

  1. Install the required R packages. Within R type

    install.packages("Rcpp", "RcppArmadillo", "RcppDE")

  2. Define external library flags in your terminal 

    Rcpp_CXXFLAGS=`Rscript -e 'Rcpp:::CxxFlags()'`
RCPP_LIBS=`Rscript -e 'Rcpp:::LdFlags()'`
RCPP_ARMADILLO=`Rscript -e 'RcppArmadillo:::CxxFlags()'`

    and export them 

    export PKG_CPPFLAGS="${RCPP_ARMADILLO} ${RCPP_CXXFLAGS}"
export PKG_LIBS="-larmadillo -llapack  ${RCPP_LIBS}"

  3. Move to the directory where RcppGO is located

    cd /path_to_RcppGO/

  4. Compile the package with R

    R CMD INSTALL RcppGO

  5. Use RcppGO Load the package in R via

    library(RcppGO)

    and open the help pages

    help("RcppGO")
